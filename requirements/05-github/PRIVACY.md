# Privacy Policy Draft

Corvus Mark v1.0 is local-first and BYOK.

## Data Sent to AI Providers

Only the following fields are sent:

- bookmark id
- title
- sanitized URL
- current path

URL sanitization removes query and hash by default.

## Data Not Sent

Corvus Mark does not send:

- API Key to Corvus Mark-owned servers
- cookies
- browsing history
- page body
- raw URL query/hash
- full bookmark tree by default

## API Key

API Key is stored locally in browser extension storage.

## Telemetry

Corvus Mark v1.0 has no telemetry, analytics, or Corvus Mark-owned backend.
